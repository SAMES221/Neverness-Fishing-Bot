# 🎣 NTE Fishing Bot v2.8

Бот для автоматической рыбалки в игре **Neverness to Everness (NTE)**.

---

## ✨ Возможности

- 🔍 **Автоматический поиск области** — бот сам находит целевую зону на экране
- 🟡 **Слежение за полоской** — автоматическое нажатие A/D для удержания полоски в зоне
- ⌨️ **Авто-нажатие F** — автоматически нажимает клавишу F для начала рыбалки
- 🖱️ **Автокликер** — автоматический клик левой кнопкой мыши (настраивается)
- 🧠 **AI режим (Kalman Filter + PID)** — фильтрация шума и упреждающее управление
- 🎨 **Кастомизация интерфейса** — меняй цвета программы под себя
- 🐟 **Умный счётчик рыбы** — считает только успешные рыбалки (по проценту попаданий в зону)
- 🔒 **Блокировка настроек** — во время работы бота нельзя изменить параметры
- ⌨️ **Настраиваемая горячая клавиша** — выбери любую клавишу для старта/стопа
- 👁️ **Предпросмотр** — встроенное окно для настройки зоны захвата

## 🚀 Как использовать

1. **Запустите файл от имени администратора** (это важно!)
2. Откройте игру Neverness to Everness и начните рыбалку
3. Настройте область захвата (вкладка Settings) или используйте автоопределение
4. Нажмите **F8** (или вашу клавишу) для запуска бота
5. Бот сам начнёт следить за полоской и нажимать нужные клавиши

---

## ⚙️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **F8** (настраивается) | Запуск / Остановка бота |

## 🧠 AI режим (Kalman Filter + PID)

Вкладка **AI**:

- **Фильтр Калмана** — убирает шум детектора и предсказывает положение курсора
- **PID регулятор** — плавное управление вместо резких движений
- **Умный счётчик** — рыба засчитывается только если вы провели в зоне достаточно времени

Все параметры настраиваются в реальном времени.
