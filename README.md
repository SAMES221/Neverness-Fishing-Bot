# 🎣 NTE Fishing Bot

Бот для автоматической рыбалки в игре Neverness to Everness (NTE).

## ✨ Возможности

- 🔍 **Автоматический поиск области** — бот сам находит целевую зону на экране
- 🟡 **Слежение за полоской** — автоматическое нажатие A/D для удержания полоски в зоне
- ⌨️ **Авто-нажатие F** — автоматически нажимает клавишу F для начала рыбалки
- 🖱️ **Автокликер** — автоматический клик левой кнопкой мыши каждые 5 секунд
- 👁️ **Предпросмотр** — встроенное окно для настройки зоны захвата
- 🎯 **Настраиваемая зона** — можно вручную настроить область слежения

## 📥 Скачать

Перейдите в раздел **[Releases](https://github.com/SAMES221/NTEFishingBot/releases)** и скачайте последнюю версию `NTEFISHBOTv1.1.exe`

## 🚀 Как использовать

1. **Запустите файл от имени администратора** (это важно!)
2. Откройте игру Neverness to Everness и начните рыбалку
3. В настройках бота настройте область захвата 
4. Нажмите **F8** для запуска бота
5. Бот сам начнёт следить за полоской и нажимать нужные клавиши

## ⚙️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **F8** | Запуск / Остановка бота |

## 🛠️ Настройка

### Область захвата
- Вкладка **Settings** → можно настроить координаты вручную
- Зелёная рамка показывает текущую зону захвата


## 📝 Системные требования

- Windows 10 / 11
- Python 3.7+ (только для запуска из исходного кода)
- Разрешение экрана: любое

## ⚠️ Важно

- **Запускать ТОЛЬКО от имени администратора!** (ПКМ → Запуск от имени администратора)
- Антивирус может ложно сработать на программу (обычное дело для ботов)
- При первом запуске может потребоваться добавить в исключения антивируса

## 🐛 Известные проблемы

- Иногда автоопределение области может работать некорректно — попробуйте настроить область вручную
- Если полоска не отслеживается — проверьте цвета в игре
